## âœ¨ Features

* **Smart Timer:** Track work remaining based on Full-Time (8h 15m) or Half-Time (4h 15m) shifts.
* **Automatic ETA:** Real-time calculation of when your workday will end, including break offsets.
* **Dev Scratchpad:** A built-in markdown-friendly notes area for snippets and todos (Auto-saved).
* **Activity Logging:** Every punch-in, break, and meeting is logged into a local SQLite database.
* **Standup Generator:** One-click to copy a formatted summary of your day's activity to the clipboard.
* **Auto-Updates:** Seamless background updates delivered via GitHub Releases with real-time UI progress.
